## Descripción

Este programa en Python permite analizar circuitos eléctricos en serie o en paralelo. El usuario ingresa las resistencias y el voltaje, y el sistema calcula la resistencia equivalente, la corriente total y los valores de voltaje o corriente en cada resistor.

Es una herramienta básica para practicar conceptos como la Ley de Ohm y el comportamiento de los circuitos eléctricos.
